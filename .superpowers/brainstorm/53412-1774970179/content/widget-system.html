Widget 系统设计

刘海左右两侧各自独立配置，每个 Widget = 数据源 × 显示风格

每个 Widget 的两个维度
① 数据源
服务：Claude / OpenAI / Gemini…
指标：
 · 剩余时间（Claude 5h窗口）
 · 重置倒计时
 · Token 用量/剩余
 · 金额余额（OpenAI）
 · 当前 Session 消耗
 · 使用百分比
② 显示风格
· 圆环进度
· 横向进度条
· 纯文字标签
· 聚合数字
配置示例
圆环 + Claude 剩余时间 → 🟡 4.2h
进度条 + OpenAI 余额 → ▓▓▓░ $18.5
文字 + 重置倒计时 → 重置 2h 14m
聚合 + Session 消耗 → ↑ 1,523 tok
